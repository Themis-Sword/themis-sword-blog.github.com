2. 安裝Octopress

確保安裝了git，在終端輸入git —version可以看到計算機中的git版本。
利用命令將octopress從github上clone到本機：
$ git clone git://github.com/imathis/octopress.git octopress
$ cd octopress
$ ruby --version
安裝相關依賴項：
$ sudo gem install bundler #需要輸入root密碼
$ bundle install
